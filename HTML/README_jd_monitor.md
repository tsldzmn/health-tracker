# 京东低价监控工具

## 安装依赖

```bash
pip install requests
```

## 使用方法

```bash
python jd_monitor.py
```

## 功能说明

### 1. 批量检查
一次性检查多个商品的当前价格，找出疑似低价商品

### 2. 持续监控
自动定时检查价格变化，发现异常低价时报警

## 如何获取SKU

1. 打开京东商品页面
2. 看URL中的数字部分
3. 例如：`https://item.jd.com/100012043978.html`
4. SKU就是：`100012043978`

## 低价判断条件

1. **价格为0或负数** - 明显的Bug
2. **低于原价30%** - 可能是优惠券叠加
3. **元变分** - 原价50元，现价0.5元，可能是单位错误
4. **价格暴跌** - 和历史价格对比，突然大幅下降

## 注意事项

1. 请求频率不要太快，建议间隔30秒以上
2. 京东有反爬机制，频繁请求可能被封IP
3. 本工具仅供学习研究使用
4. 发现的低价商品可能被砍单
